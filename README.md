# A Casa Sempre Ganha

Simulador de Roleta desenvolvido em React.js para validar estatisticamente a Teoria da Ruína do Jogador. O projeto demonstra matematicamente como, no longo prazo, a vantagem da casa leva inevitavelmente à perda do capital do apostador.

## Tecnologias
* **React.js** (Hooks: `useState`, `useEffect`, `useRef`)
* **CSS3** (Animações e Gradientes Cônicos)
* **SVG** (Renderização gráfica nativa)

## Como Rodar
1.  Clone o repositório:
    ```bash
    git clone [https://github.com/seu-usuario/a-casa-sempre-ganha.git](https://github.com/seu-usuario/a-casa-sempre-ganha.git)
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o projeto:
    ```bash
    npm start
    ```
    Acesse em `http://localhost:3000`.

## Funcionalidades
* **Roleta Interativa:** Giro com física realista e alinhamento orbital dos números.
* **Gráfico em Tempo Real:** Visualização da flutuação do saldo (Verde = Lucro, Vermelho = Prejuízo).
* **Simulação de Monte Carlo:** Botão para simular 100 rodadas instantaneamente e provar a tendência de ruína.
